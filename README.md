# Data Engineering Basics


This repository contains a collection of Python notes, code examples, and assignments covering fundamental concepts for data engineering. The material is structured to provide a step-by-step learning path, from basic syntax to more advanced topics like Object-Oriented Programming and data serialization.

## Repository Structure

The repository is divided into several modules, each focusing on a specific set of Python concepts.

### `Python-Basics-01`
This module introduces the absolute fundamentals of Python.
- **Topics:** Variables, Data Types (`int`, `float`, `str`, `bool`), Operators (Arithmetic, Relational, Logical), and handling standard Input/Output.

### `Python-Basics-02`
This section covers control flow and functions, the building blocks for writing structured programs.
- **Topics:** Conditional statements (`if`/`elif`/`else`), Loops (`while`, `for`), `range()`, `break`/`continue`, function definitions, parameters, arguments, and the `match` statement.

### `Python-Basics-03`
A deep dive into Python's core data structures.
- **Topics:** In-depth exploration of Strings, Lists, Tuples, Dictionaries, and Sets, including their methods, indexing, slicing, and common operations. Contains assignment problems and solutions.

### `Python-Basics-04`
This module focuses on Object-Oriented Programming (OOP) principles in Python.
- **Topics:** Classes and Objects, Constructors (`__init__`), Attributes (Instance vs. Class), Methods (Instance, Class, Static), Encapsulation, Inheritance, Abstraction, and Polymorphism. Includes a practical chat system example demonstrating these concepts.

### `Python-Basics-05`
Covers essential topics for file and data handling.
- **Topics:** File I/O (reading, writing, appending), Exception Handling (`try`/`except`/`else`/`finally`), List Comprehensions, and working with the JSON module for data serialization and deserialization.

## Key Concepts Covered

- **Python Fundamentals:** Variables, data types, operators, I/O.
- **Control Flow:** `if-else` statements, `for` and `while` loops.
- **Functions:** Defining functions, parameters, return values, and default arguments.
- **Data Structures:** Strings, Lists, Tuples, Dictionaries, and Sets.
- **Object-Oriented Programming (OOP):** Classes, objects, inheritance, encapsulation, polymorphism, and abstraction.
- **File Handling:** Reading from and writing to files using various modes.
- **Exception Handling:** Managing runtime errors gracefully with `try-except` blocks.
- **Data Serialization:** Reading and writing data using the `json` module.
- **Advanced Techniques:** List comprehensions for concise and readable code.

## How to Use

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/harjotsingh17/data_engineering_basics.git
    ```
2.  **Navigate the directories:**
    It is recommended to go through the `Python-Basics-XX` folders in sequential order.
3.  **Review the notes:**
    Each folder contains `.md` files or Jupyter notebooks (`.ipynb`) with detailed explanations.
4.  **Run the code:**
    Execute the `.py` scripts or run the cells in the `.ipynb` notebooks to see the concepts in action.
